03 / Projects

Things I've built

🌾
Featured project

KrishiBondhu

KrishiBondhu ("Farmer's Friend") is a comprehensive full-stack MERN agricultural ecosystem designed to bridge the gap between Bangladeshi farmers, agricultural experts, and bulk buyers. Built with three distinct role-based portals — farmers get real-time weather intelligence, AI-powered crop recommendations, a disease detection library, and live market price updates; buyers get a verified marketplace with order management and procurement analytics; and admins get a live dashboard to manage the entire platform. Integrates OpenWeatherMap, Cloudinary, and JWT authentication with a fully responsive custom CSS UI.

React 19 Vite Node.js Express 5 MongoDB Atlas JWT Cloudinary OpenWeatherMap API
🗺️

TrekWise

A smart cross-platform travel planner built with Flutter and Firebase over the course of 2025, designed for navigating fragmented transport systems in developing regions like Bangladesh. Lets users compare multi-modal routes by speed, cost, or convenience; discover community-verified local guides and travel packages; and access real-time weather alerts. Features offline route saving for low-connectivity areas, expense tracking, and a clean mobile-first UI — fully compatible with Android and iOS.

Flutter Dart Firebase OpenStreetMap API OpenWeatherMap API
🏹

Goblin's Curse

A 2D action-adventure game where Milo, a treasure hunter, battles relentless waves of goblins using a Magic Bow to claim the Goblin King's legendary cursed treasure. Built from scratch in C++ with the iGraphics/OpenGL library — featuring a two-level campaign set across an Enchanted Forest and Ancient Cave, progressive enemy difficulty scaling, a special power-up system (collect 5 energy bars to unlock a multi-kill super arrow), and custom bitmap rendering, collision detection, and background music.

C++ C OpenGL iGraphics Visual Studio 2D Game Dev
